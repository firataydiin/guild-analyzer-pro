# ⚔️ Guild Analyzer Pro

**Excel tabanlı bir guild analiz sistemi.**  
Kullanıcı girişi olan, verileri kalıcı olarak saklayan ve web tarayıcısından erişilebilen bir Streamlit uygulamasıdır.

## 🪶 Özellikler
- 🔐 Kullanıcı girişi ve kayıt (şifreli)
- 📊 Excel dosyası yükleme
- 🧮 Might ve Kill farkı analizi
- 🗑️ Satır silme (admin onayıyla)
- 📈 Görsel istatistik dashboard’u
- 💾 Kalıcı SQLite veritabanı (`data.db`)

## 🚀 Çalıştırma (lokal)
```bash
pip install -r requirements.txt
streamlit run app.py
